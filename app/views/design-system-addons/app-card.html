App card

Use app card to group clinic details into compact sections on the change clinic summary pages.

When to use this component

  • Change series and change single clinic summaries
  • Sections like capacity, clinic times, services, closures, and one-off changes
  • Rows that may include formatted html and row actions

Options

Option Type Description
title string Card heading text.
action object Action link in header with href, text and visuallyHiddenText.
items array Rows rendered as summary list key/value pairs.
html string Custom body content. Used instead of items when provided.
classes string Additional classes on the root card element.
attributes object Custom html attributes applied to the root card element.
