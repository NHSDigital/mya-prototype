## Why this journey exists

The clinics list is the home base for managing a site's schedule. It shows recurring series and
one-off clinics together, with the services each offers, so a user can find the clinic they need
to change or cancel.

## Key decisions

- The list paginates rather than loading everything at once, keeping long histories manageable.
- Each row links straight into the change-clinic journey, so editing is one click away.
